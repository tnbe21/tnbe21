## 建築業界向けSaaS（正社員）
### 2023年10月 - 現在
* 職種： SaaS開発エンジニア
* 業務内容:
  * SaaS新機能開発・既存機能改修における基本・詳細設計(DBスキーマ設計、詳細仕様書作成等)
  * SaaSプロダクト = マイクロサービス構成のReact, Node.js(Webフロント、バックエンド問わずアプリ全般の)実装とレビュー
  * 各環境へのリリース(ソースコードの配備、DB更新(スキーマ適用やバッチ実行等))
  * 実装やそのリリースにかかるワークフロー(gitサブモジュール更新自動化、ワークフロー実行時のslack通知処理等)の整備
  * テストケースとテスト結果のレビュー
  * 日時定例での各メンバーの作業進捗確認、また主にQAチームとの各環境へのリリース日程調整
* 実績・取り組み:
  * ユニットテスト効率化
    * 以前はリポジトリ全体でしか実行できなかったのを、ユニットごとに実行できるよう改善
    * 開発メンバーの手元に加え、Github上でも実行できるよう整備した
  * 期間3, 4ヶ月(10人月)ほどの中規模案件の開発リード
    * DBスキーマ設計、プロダクトオーナーと仕様書の詳細を調整(リファインメント)
    * 社外に提供するAPIを別途実装するチームと設計を共有、仕様調整、負荷試験作業補助、日程調整
    * コードのディレクトリ構成等含む、開発メンバーのコードレビュー
    * 複数PRを商用ブランチにマージした上でのリリース作業実施
  * コードレビューやテストレビュー等技術面でのレビュー経験
  * メンバーに実装依頼(設計・仕様書の説明)した上でチームとして開発やそのフロー整備を推進する経験
  * TypeScriptやGraphQL、Reactの実装・コードリーディング力
  * Docker仕様面の理解: 複数台構成(compose)、キャッシュ機構 等
  * Notionをベースにした開発の進め方
  * スクラム開発知識習得
* 技術スタック：
  * TypeScript
  * GraphQL
  * MySQL
  * MongoDB
  * AWS（ECS, S3, CloudFront）

  * GitHub Actions
